# Auto-complétion intelligente pour scénarios d'ingénierie des exigences

Stage Licence 3 - IRIT SMAC  


## Objectif
Développer un outil d'auto-complétion générique pour l'assistance 
à l'écriture de scénarios, avec priorisation du vocabulaire du glossaire.
